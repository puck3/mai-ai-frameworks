# Прикладные системы и фреймворки искусственного интеллекта

## Классификация

Датасет: Employee dataset (бинарная классификация).

Метрика: ROC AUC

В задаче бинарной классификации используется метрика ROC AUC, поскольку она:
- позволяет сравнивать модели независимо от выбора порога классификации;
- устойчива к дисбалансу классов.

## Регрессия

Датасет: Calories Burnt Prediction.

Метрика: MAPE

Для задачи регрессии выбрана метрика MAPE (Mean Absolute Percentage Error), так как она:
- не зависит от масштаба целевой переменной;
- интерпретируется в процентах и удобна для сравнения моделей.

## Результат

Итоговый вывод:

1. Ансамблевые методы (градиентный бустинг и случайный лес) показали наилучшее качество как в классификации, так и в регрессии.

2. Линейные модели продемонстрировали худшие результаты и могут рассматриваться только как базовый ориентир.

3. Самописные реализации уступают sklearn по скорости и качеству, но корректно воспроизводят поведение алгоритмов и подтверждают теоретические ожидания.
